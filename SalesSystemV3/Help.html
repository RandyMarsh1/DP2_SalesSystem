Sales Database

1. Adding a sales record

To add a sales record, you must first add items into the sale. The total cost of the sale will be automatically calculated based on the quantity of the items included in the sale.

New items that were not previously listed in the inventory will be automatically added to the inventory.



2. Updating a sales record

To update an existing sales record, simply click anywhere on the row in the database's table display.

You can then edit the values in each column individually or use the EDIT SALES RECORD form located to the right of the window.

Once you are satisfied with your changes, press the UPDATE SALE button on the EDIT SALES RECORD form.


Please note that your changes will not be made to the database unless the UPDATE SALE button has been pressed.



3. Removing a sales record

To remove an existing sales record, simply click anywhere on the row in the database's table display.

You must then click on the DELETE SALE button in the EDIT SALES RECORD form located to the right of the window.



Inventory Management

4. Adding new items

To add a new product to the inventory, you must use the ADD NEW ITEM TO INVENTORY form located to the right of the window.

You must specify its ID number, name, a description (optional), the product's type, and cost/sale prices.


5. Updating existing items

To update an existing product already listed in the inventory, simply click anywhere on the row in the database's table display.

You can then edit the values in each column individually or use the EDIT SELECTED ITEM form located to the right of the window.

Once you are satisfied with your changes, press the UPDATE button on the EDIT SELECTED ITEM form.


Please note that your changes will not be made to the database unless the UPDATE button has been pressed.



6. Removing existing items

To remove an existing product, simply click anywhere on the item's row in the database's table display.

You must then click on the DELETE button in the EDIT SELECTED ITEM form located to the right of the window.



Sales Reporting

7. Searching the database

Clicking on the box next to the SEARCH label will allow you to search the database for a specific set of sales.

You can adjust the parameters of the search by changing the range of the dates to search from (based on a monthly basis only).


8. Exporting the database

Clicking the EXPORT CSV button will enable you to export the contents of the entire database to a directory of your choice on the local machine.

Select the folder in which you wish to export the file to. The software's file handler will convert all the records into a Comma Separated Values (.csv) file that can be accessed by text editors.


Please note that the exported data will only range from the dates specified in the SEARCH section. If you wish to export the entire database, then leave the former field blank.



Prediction Reporting

9. Prediction

The system will automatically generate a report based on the sales records of products in the database.

If there insufficient data then the system will not be able to generate a report.
